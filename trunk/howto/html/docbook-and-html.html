4.2. DocBook 和 HTML 语言

正如您前面所见到的，DocBook 和 HTML 文档的结构类似。如果您已经习惯了 HTML 语言，您可能会觉得 DocBook 的标记符十分的冗余。DocBook 也包含一些逻辑标记符，而没有（或者很少）使用像 HTML 中那些版式控制标记符。DocBook 的这种理念使得我们在撰写文档的时候能够尽可能的提供更多的信息，以便其它的软件能够对其进行更多的控制。在 DocBook 中，我们使用标记符标识的是各部分的含义，而不是它们陈述的内容。

我们使用 DocBook 的原因，还包括它减轻了我们在生成文档的某些输出格式中的工作，它扮演着我们需要在文档中所要表达的一切内容的中心角色。

您不能在 DocBook 中创建新的标记符，而只能使用它所提供的标准标记符集。在翻译工作中，请不要翻译标记符，因为他们需要和英语版本的文档保持一致，以便文档的维护工作能够正常的进行。

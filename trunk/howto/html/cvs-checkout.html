3.3. 模块校验

现在，您可以校验（即下载）phpdoc 模块了。所谓模块，是指一系列 的源文档以及目录，通常它们就是 CVS 资源库的一个子目录。

用 cd 命令切换到您希望用来储存 phpdoc 模块 的目录，执行以下命令，将在该目录下建立一个名为 phpdoc 的子目录: 

$ cvs checkout phpdoc

当系统显示完一些校验命令的回显信息后，您就应该已经拥有了一个完整的 phpdoc 模块了。

Note: 在决定将 phpdoc 目录放在哪里的问题上，您并非是完全自由的。请查 阅“工具及安装须知”一节以获更准确的信息来确认 该 phpdoc 目录在您的系统中是否受到限制。

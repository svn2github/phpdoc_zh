第 1 章. PHP 文档的框架

PHP 文档是用 XML 语言编写的，同时运用了 DocBook DTD。如果您希望参与 PHP 文档工程的工作，那么您至少需要了解有关 XML 和 DocBook的基本知识。

XML 文档是保存在中央服务器的，并且可以用 CVS 的客户端来访问。现在有许多不同的 CVS 客户端可用，我们推荐您使用最简单的命令行工具。

您需要用其它的程序和工具来操作 XML 文档以及测试这些文档内容的错误。您需要什么样的工具，取决于您使用的操作系统。尽管与 PHPDOC 相关的许多工作都可以在 Windows 下进行，我们仍然推荐您使用 Linux 或者某种形式的 Unix 系统。您可以在“工具及安装须知”一节找到有关您所需要的工具的更多信息。

如果您遇到了本文档未涉及的问题，请发电子邮件到 <phpdoc@lists.php.net> 询问。

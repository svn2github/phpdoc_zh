3.4. 更新目录树

我们建议您随时更新您的 phpdoc 目录树以保证您拥有最新的文件。目前，该目录下的内容几乎每天都有更新，因此，在您进行编辑工作之前，请一定要先更新您的文件。

要更新您的文件，请用 cd 切换到 phpdoc 目录并运行如下命令： 

$ cvs update

注意： 如果您下载了任何版本的翻译模块，那么以上命令将自动更新其模块的内容。因此您没有必要单独更新翻译模块下的文件。

如果您仅希望更新一个或某一些您需要更改的文件，您可以在 update 命令后面加加上它们的文件名： 

$ cvs update file1 [file2]

如果这些文件不在 phpdoc 的根目录下，请加上这些文件的相对路径： 

$ cvs update en/functions/file1 [en/chapters/file2]

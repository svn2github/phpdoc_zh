第 14 章. 查看您的更新

我们已几种不同的格式输出文档的可阅读版本，其中最有用的应该是：

  • html，普通 HTML 版本，请查看 http://www.php.net/manual/html/

  • bightml，单个大型 HTML 版本， 该版本仅在以下地址提供下载：http://www.php.net/download-docs.php

  • phpweb，PHP 在线手册版本，请查看 http://www.php.net/manual/

  • pdf，PDF 版本的手册，请查看 http://www.php.net/distributions/php_manual_en.pdf

  • rtf，RTF (Rich Text Format) 格式的版本，目前尚未在线发布。

  • howto，即本指引文档的普通 HTML 版本，请查看 http://cvs.php.net/co.php/phpdoc/howto/howto.html.tar.gz，其简体中文版本可以到 http://cvs.php.net/co.php/phpdoc-zh/howto/howto_zh_cn.html.tar.gz 下载。

要生成可阅读版本的文档，请用 cd 命令切换到 phpdoc 目录，并运行以下命令中的对应项： 

$ make html
$ make bightml
$ make phpweb
$ make pdf
$ make rtf

要生成本指引（HOWTO）文档： 

$ make howto

用 lynx 过滤 bightml 格式，即可得到 PHP 文档纯文本格式的版本。

注意： 可阅读版本的文档的生成，是一个很占用 CPU 的过程，这个过程将花费较长的一段时间。比如，在 700MHz 主频的机器上，生成普通 HTML 格式的手册需要大概 7 分 30 秒。

输出文件存放的位置取决于您生成的格式。 HTML 手册将被生成到 phpdoc 目录的 html 子目录下，同理 make phpweb 命令将把文件存放到 php 目录中。指引文档（HOWTO）的 HTML 文件将被存放到 howto/html 目录下。

如果一切都进行的很顺利，您将得到一份包括您所作的更新的 PHP 文档。

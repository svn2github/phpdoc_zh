第 12 章. 关于 Makefile 文件

为 PHP 文档所作的贡献中，一个很重要的部分，是我们能够在使您的更改生效后，以各种格式查看文档输出的结果，包括 HTML、Adobe PDF、RTF 以及其它格式。

Makefile 文件控制着文档生成和生效的几个重要方面。要从一个新的 checkout 模块中生成 Makefile 文件，请用 cd 命令切换到 phpdoc 的目录并执行一下命令： 

$ autoconf

$ ./configure

该命令将生成您的 phpdoc 目录树的当前 Makefile 文件。

注意： 如果您使用的是 Windows 操作系统，您需要 Cygwin 的 bash 外壳来执行这些命令。

在设置项中，您可能需要用到两个重要的参数。一个是 --with-dsssl。使用该参数，您可以指明 DSSSL 样式表单的路径。系统将会搜索很多（15个以上）目录查找 DSSSL，因此您不必太在意这个参数。同时。您可以用 --with-lang 参数来指明手册使用的语言，其默认值是“en”。以下是使用该参数的一个范例： 

$ ./configure --with-lang=zh

另外还有很多其它的参数，其中比较重要的，有 --with-xsl 参数，可以用来指明 XSL 样式表单的路径，以及 --with-zendapi 参数，可以指明 ZEND API 文档的路径。
